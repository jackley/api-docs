# Get Group

Get the information of the specified user group.

## HTTP Request

```bash
GET /api/1/groups/[id]
```

## Example Request

```bash
$ curl https://database.account.directus.io/api/1/groups/1
```

```javascript
client.getGroup(1);
```

{
  "id": "1",
  "name": "Administrator",
