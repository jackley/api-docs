# Get Tables

Get the list of tables viewable by current user

> **Note:** Table names are case-sensitive

## HTTP Request

```bash
GET https://database.account.directus.io/api/1/tables
```

## Example Request

```bash
$ curl https://database.account.directus.io/api/1/tables \
        -u [user-token]:
```

```javascript
client.getTables();
```

## Response
```json
[
  {
    "table_name":"projects"
  },
  {
    "table_name":"articles"
  },
  {
    "table_name":"clients"
  }
]
```
